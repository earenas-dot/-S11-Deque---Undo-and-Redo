# Sistema Undo/Redo con Deque en Python

## Descripción

Este proyecto implementa un sistema de edición con funcionalidades de:

- Agregar acciones
- Undo
- Redo
- Historial de acciones
- Estado actual

El objetivo principal es aplicar la estructura de datos Deque en un problema práctico utilizando Python y una interfaz gráfica.

---

# Tecnologías utilizadas

- Python 3
- Tkinter
